Um simples projeto para gerenciar reuniões diárias da metologia ágil Scrum.

Sobre o Projeto

As reuniões diárias do Scrum, conheciadas como Daily, são reuniões qure ocorrem todos os dias com intuito de cada membro da equipe informa como anda o seu trabalho. O SGRDS vem para facilitar esse trabalho através de formulários para criar registros da daily e relatórios para vizualizar esses registros.

Tecnologias Utilizadas

O projeto foi divido em duas partes, sendo elas:

Front-end

Parte visual do site, responsável pela interação com usuário. Dentre as tecnologias usadas temos:
  • Angular: Proderoso framework Front-end desenvolvido originalmente pela google, sendo hoje uma das tecnologias mais utilizadas no lado cliente da aplicação.
  • TypeScript: Linguagem de programação que adiciona tipagem forte ao JavaScript em tempo de produção.
  • Bootstrap: Um Framework Front-end que tem como foco a estilização do site usando de regras CSS e JavaScript para iteração.

Back-end

Parte da aplicação responsável por ser execultada no servidor e interagir com o banco de dados.
  • Spring Boot: Robusto Framework Back-end escrito em Java que facilita o desenvolvimento de aplicações em Spring.
  • MySQL: Banco de Dados Relacional comumente utilizado na web que usa da linguagem SQL para realizar consultas e comandos ao banco de dados.
